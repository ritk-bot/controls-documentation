To restore a certain file to the version on github
```
git restore <path_to_file>
```
and to check the differences of the current file from the last pushed commit
```
git diff <path_to_file>
```

Apparently gz sim uses some sort of bridge to convey its clock to ros which was not present in my sim.launch.py.

```
Node(

package="ros_gz_bridge",

executable="parameter_bridge",

arguments=[

"/clock@rosgraph_msgs/msg/Clock[gz.msgs.Clock"

],

output="screen",

),
```

The biggest pointer for this is checking /joint_states:

```
ros2 topic echo /joint_states --once
```

and checking /clock and seeing whether it has any publishers:

```
ros2 topic info /clock

```
YAMLs are case sensitive pls check even small spaces like 
```
-gripper
```
